Projects

Below are some of the projects that I have worked on!

REDACTED - Autonomous Maze Explorer

REDACTED is a robotic system capable of autonomously exploring and traversing an unknown maze. The robot is able to find the position of all obstacles, determine the fastest path from START to END, as well as perform image recognition of signs in the maze. In addition, the robot can be controlled wirelessly through an Android app.

ParkLah - Driver's Mobile Assistant

ParkLah is a one-stop mobile application that aims to help regular, everyday drivers solve everyday challenges that they might face when driving. By providing clear, relevant information about amenities in and around the intended destination, the driver may make informed decisions on their commutes. Drivers are also able to request help and guidance in the event of emergencies on the road.
